# MyZap 🌟

MyZap 是一个基于 Zap 封装的更好用的日志库，提供了以下功能：📝

1. 自定义日志格式 ✨
2. 支持日志选项参数 🛠️
3. 支持将不同级别日志输出到不同位置 📂
4. 支持日志按大小和时间进行轮转 🔄
